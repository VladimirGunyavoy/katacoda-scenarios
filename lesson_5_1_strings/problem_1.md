Выбирая отдельные символы из строки, можно составлять слова. Вот пример:

```
my_string = 'Съешь ещё этих мягких французских булок, да выпей чаю.'

print(my_string[0])
print(my_string[2])
print(my_string[15])
print(my_string[16])
```

Вывод:

```
С
е
м
я
```

Выведите слово «мягкий», как показано в примере.
Вывод:
```
м
я
г
к
и
й
```
